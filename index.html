3D Printed Horn Array Antenna (2x1)

This was an RF project where I 3D printed a horn antenna. The calculations for the specifications for the horn antenna were done using numerical methods with a program I wrote in Python. After 3D printing the antenna, a single layer of copper tape was applied after account for skin depth. A homemade desktop CNC milled 3dB hybrid coupler was also created. Microwave Office was used for the initial design of the coupler, HFSS was then used as a FEM tool to verify. This was then exported to a Gerber file for fabrication but due to production delays, a desktop CNC was used which required GCode.
